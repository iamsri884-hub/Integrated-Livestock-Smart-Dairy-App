# Backend API

This folder contains backend API design for the Integrated Livestock & Smart Dairy App.

## Planned APIs

### Authentication
- POST /login
- POST /register

### Livestock
- POST /add-livestock
- GET /livestock
- PUT /update-livestock
- DELETE /delete-livestock

### Milk Production
- POST /add-milk
- GET /milk-records

### Milk Sales
- POST /add-sale
- GET /sales

### Dairy Products
- POST /add-product
- GET /products

### Market Prices
- POST /update-price
- GET /market-prices
